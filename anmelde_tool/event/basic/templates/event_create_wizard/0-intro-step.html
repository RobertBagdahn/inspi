Wie funktioniert der Wizard?

Der Wizard führt Sie durch 7 einfache Schritte zur Erstellung einer neuen Veranstaltung:

  1. Einführung (hier)
  2. Grundlegende Informationen
  3. Veranstaltungsort
  4. Zeitplanung
  5. Teilnehmer
  6. Berechtigungen
  7. Zusammenfassung

Tipps

  • Sie können zwischen den Schritten hin und her navigieren
  • Pflichtfelder sind mit einem * gekennzeichnet
  • Halten Sie die Maus über ein Infoicon (i), um weitere Informationen zu erhalten
  • Nach dem letzten Schritt wird die Veranstaltung erstellt
